# Typography

The type scale from `src/styles/tokens.css`, set in **Geist** (`font-sans`). Each token
defines a font size and a paired line height — values below come straight from the tokens.

Weight is a **separate axis**: it isn't baked into the size tokens, so pick a size class
(e.g. `text-body-14`) and combine it with a Tailwind weight utility (see the bottom of this
page). For example a "Body / 14 / Semi Bold" style is `text-body-14 font-semibold`.

## Caption

Metadata, labels, and helper text.

<div>

Combine any size above with one of these. Geist ships these weights; values are Tailwind
defaults (not tokens).
